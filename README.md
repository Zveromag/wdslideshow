# WDSlideShow

## Description

This plugin is a slide show that can be used in any project.

## Initialization

````javaScript
new WDSlideShow();
````

## Options

defaults values

````javaScript
arrows: true, //show arrow
caption: true, //show caption
keyboard: true, //use keyboard controls
onChange: function() {} //change slide event
````
